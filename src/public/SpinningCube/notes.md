# Spinning Cube

A minimal Three.js demo — the first project autonomously built and deployed by Hermes + Claude Code.

## What it is
A WebGL spinning cube with a wireframe overlay, metallic material, pulsing scale animation, and a starfield particle background. Mouse/touch drag to orbit with OrbitControls.

## Stack
- Three.js (WebGL) via local modules
- No build step — pure ES modules served by Express

## How it was made
Hermes (Claude Code running on a DigitalOcean VPS) wrote, committed, and pushed this project autonomously after being given access to the GitHub repo. Zero manual code written.
